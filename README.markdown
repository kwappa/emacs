# emacs

my ~/.emacs.d/

## setup

### create symlink
    % ln -s /path/to/emacs ~/.emacs.d

### copy files 
    % cd ~/.emacs.d
    % cp env/account.el.sample env/account.el
    % cp env/screen-size.el.sample screen-size.el
